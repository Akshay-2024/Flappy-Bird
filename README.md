**Flappy Bird Clone**

A nostalgic, browser-based tribute to the classic mobile game that defined an era.

🎮 About the Project
This project is a faithful recreation of the original Flappy Bird game. Built with simplicity in mind, it brings back the challenging "one-more-try" gameplay directly to your web browser. The goal is simple: navigate the bird through as many pipes as possible without hitting them or the ground.

This was built primarily for entertainment and to demonstrate how a complete game loop can be achieved using lightweight web technologies.

✨ Features
Simple Mechanics: Easy-to-learn, hard-to-master gameplay.

Physics Engine: Custom gravity and collision detection logic.

Responsive Design: Playable on various screen sizes.

Score Tracking: Real-time scoring to keep track of your progress.

🛠️ Tech Stack
HTML5: Structured the game container and UI elements.

CSS3: Handled the styling, animations, and layout.

JavaScript: The engine behind the game logic, physics, and movement.
